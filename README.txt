gmail_robot.txt :- Gmail test automation using selenium2library and Robot automation framework
		   Test Cases:
			1] Positive Gmail Log In Test
			2] Negative Gmail Log In Test with Wrong Password
			3] Negative Gmail Log In Test with Wrong Email Address

Execution command:- robot -d result gmail_robot.txt

Tools     :- Python 2.7.11
	     Robot Automation Framework 3.0
	     Selenium2Library 

Note      :- To solve webdriver exception error download chromedriver 2.21 (https://sites.google.com/a/chromium.org/chromedriver/downloads)
	     and keep it at ...\Python27\Scripts\ .
